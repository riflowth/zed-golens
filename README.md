# Go CodeLens for Zed

This extension enhances [Go](https://go.dev/) CodeLens support in [Zed](https://zed.dev/) by integrating with the GoLens language server.

> All CodeLens capabilities is provided by [GoLens](https://github.com/vectier/golens) itself. This extension is responsible only for downloading the GoLens binary and launching it as a language server within Zed.
